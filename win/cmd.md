## 进入cmd的方法
* 资源管理器中的路径中输入cmd进入cmd,路径是当前路径
* 开始菜单输入cmd
* win+R打开运行，输入cmd

    netsh wlan show profile                                          #查看wlan名称
    netsh wlan show profile name="wifi名称" key=clear                #查看wlan密码


    pause                                                            #暂停

   ping -t ip                                                        #不停地访问指定ip,ctrl+c后统计发送信息
